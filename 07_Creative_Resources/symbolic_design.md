# Symbolic Design Knowledge

## Symbol Categories

### 1. Geometric
- **Circles**: wholeness, completion, unity
- **Spirals**: growth, evolution, journey
- **Triangles**: transformation, ascension, trinity
- **Squares**: stability, foundation, structure
- **Lines**: direction, connection, separation

### 2. Natural
- **Trees**: growth, roots, life
- **Water waves**: emotion, flow, change
- **Mountains**: challenges, achievement, permanence
- **Fire**: transformation, passion, destruction/creation
- **Wind/Air**: freedom, thought, spirit

### 3. Mythic
- **Runes**: ancient wisdom, mystery
- **Sigils**: intention, magic, manifestation
- **Archetypal icons**: universal symbols
- **Sacred geometry**: divine patterns

### 4. Mechanical
- **Circuits**: connection, technology, systems
- **Gears**: interdependence, mechanics, process
- **Microstructures**: detail, foundation, building blocks

## Symbol Extraction Method

### 5-Step Process:
1. **Identify Emotional Wound**
   - What pain needs acknowledging?
   - What's the core struggle?

2. **Identify Transformation Goal**
   - What's the desired state?
   - What change is sought?

3. **Choose Metaphor**
   - What natural/mythic image captures the journey?
   - What resonates symbolically?

4. **Convert Metaphor → Visual Symbol**
   - Distill to essential visual elements
   - Create simple, memorable form

5. **Use Symbol in UX or Brand**
   - Integrate into design system
   - Apply to touchpoints
   - Create consistency
