# UX Templates

## Story-First Wireframes

### Concept:
Wireframes that prioritize emotional narrative over functionality.

### Key Elements:
1. **Emotional Entry Point**
   - Hook that connects to user's feeling
   - Resonant imagery or copy
   - Immediate emotional alignment

2. **Journey Markers**
   - Visual indicators of progress
   - Story beats throughout flow
   - Emotional checkpoints

3. **Transformation Moments**
   - Key decision points framed as story moments
   - Before/after visualizations
   - Celebration of progress

### Template Structure:
```
Screen 1: Emotional Hook
- What feeling are we invoking?
- What story are we starting?

Screen 2-4: Journey Development
- How does the user move through transformation?
- What obstacles do they overcome?

Screen 5: Resolution/Outcome
- What has changed?
- How do we celebrate?
```

## Archetypal Flows

### Concept:
User flows designed around archetype behaviors and motivations.

### Creator Flow:
- Emphasize: Freedom, tools, expression
- Minimize: Constraints, rigid templates
- Key moments: Creation, customization, sharing

### Warrior Flow:
- Emphasize: Goals, achievements, progress
- Minimize: Ambiguity, unnecessary steps
- Key moments: Challenges, victories, rankings

### Sage Flow:
- Emphasize: Learning, depth, understanding
- Minimize: Superficiality, rushed decisions
- Key moments: Discovery, insights, mastery

### Explorer Flow:
- Emphasize: Discovery, freedom, options
- Minimize: Linear paths, constraints
- Key moments: Exploration, choices, new territories

### Caregiver Flow:
- Emphasize: Helping others, impact, nurturing
- Minimize: Self-focus, competition
- Key moments: Contribution, support, community

## Emotion Safety Screens

### Concept:
Screens designed to create psychological safety at vulnerable moments.

### Components:

1. **Grounding Element**
   - Visual anchor (symbol, image)
   - Calming color palette
   - Spacious layout

2. **Affirming Copy**
   - "You're not alone in this"
   - "This is a safe space"
   - "Take your time"

3. **Clear Exit Path**
   - Always visible
   - No shame in leaving
   - Save progress option

4. **Progressive Disclosure**
   - Reveal one thing at a time
   - Don't overwhelm
   - Build trust gradually

5. **Reflection Prompts**
   - Optional journaling
   - Guided questions
   - Self-discovery tools

### Use Cases:
- Onboarding emotional work
- Vulnerable data collection
- Identity exploration
- Difficult decision points

## Symbol Kits

### Tattoo-Inspired Icons

**Style Guidelines:**
- Bold line work
- Minimalist forms
- Symbolic meaning
- Timeless aesthetic

**Icon Categories:**
- **Navigation**: arrows, paths, compasses
- **Actions**: hands, tools, gestures
- **States**: circles (complete), spirals (in progress), triangles (alert)
- **Emotions**: heart (love), flame (passion), wave (flow)
