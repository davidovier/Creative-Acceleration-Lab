# Core Workflows

## Prototype Workflow

1. **Intake**
   - Review intake form
   - Initial clarity call
   - Gather requirements and sketches

2. **Story Mapping**
   - Extract emotional core
   - Identify hero's journey
   - Map narrative structure

3. **UX Sketching**
   - Create low-fidelity wireframes
   - Design story-first flows
   - Map emotional journey through UI

4. **Technical Mapping**
   - Define data models
   - Choose architecture
   - Plan AI integrations

5. **Build**
   - Rapid prototyping
   - Daily micro-demos
   - Iterative refinement

6. **Test**
   - Emotional validation
   - Functional testing
   - User feedback loops

7. **Deliver**
   - Final presentation
   - Narrative documentation
   - Handoff and next steps

## Story Workflow

1. **Emotional Extraction**
   - Conduct deep interviews
   - Use extraction prompts
   - Identify core wounds and desires

2. **Archetype Detection**
   - Match behavioral patterns
   - Identify primary archetype
   - Consider blended types

3. **Symbol Mapping**
   - Convert emotions to visuals
   - Create metaphor library
   - Design symbolic elements

4. **Narrative Assembly**
   - Construct hero's journey
   - Build product myth
   - Create founder narrative

## Creative School Workflow

1. **Student Onboarding**
   - Welcome ritual
   - Community introduction
   - Goal setting

2. **Identity Assessment**
   - Archetype discovery
   - Emotional clarity session
   - Current state evaluation

3. **Learning Path Assignment**
   - Match modules to goals
   - Create personalized curriculum
   - Set milestones

4. **Challenges**
   - Weekly creative exercises
   - Prototype challenges
   - Peer collaboration

5. **Portfolio Creation**
   - Build showcase pieces
   - Create narrative around work
   - Prepare for presentation
