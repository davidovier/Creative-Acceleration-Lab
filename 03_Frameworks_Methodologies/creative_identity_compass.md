# Creative Identity Compass

## Framework Overview
A systematic approach to helping creatives and professionals discover and express their authentic identity.

## 5-Step Process

### 1. Archetype Discovery
- Explore core archetypal patterns
- Identify primary and secondary archetypes
- Understand how archetypes manifest in behavior and creation

### 2. Emotional Clarity
- Identify emotional drivers and wounds
- Map desire landscape
- Understand fear patterns
- Recognize emotional triggers

### 3. Symbol Creation
- Extract personal symbols from emotions
- Create visual representations
- Design symbolic identity elements
- Develop signature aesthetic

### 4. Life Prototype
- Design small experiments to test identity expression
- Create portfolio pieces aligned with identity
- Build identity-driven projects
- Test authentic expression in safe contexts

### 5. Expression Ritual
- Establish daily/weekly creative practices
- Create rituals for emotional release
- Build sustainable expression habits
- Maintain identity alignment

## Deliverables
- Archetype summary
- Emotional clarity report
- Personal symbol kit
- Life prototype plan
- Expression ritual guide

## Emotional Clarity Summary Template

### Components:
- Core wound
- Current identity
- Desired transformation
- Fear patterns
- Archetype match
- Symbolic direction

## Creative Identity Pipeline

1. Emotional Q&A → Insight Agent
2. Archetype draft → Story Agent
3. Symbol creation → Expression Agent
4. Life prototype → Mentor Agent

## Applications
- Personal branding
