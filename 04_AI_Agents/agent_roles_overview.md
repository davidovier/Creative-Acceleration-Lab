# AI Agent Roles Overview

## Core Agent Roles

### 1. Prototype Engineer Agent
**Purpose:** Build MVPs, technical docs, user stories

**Inputs:**
- Requirements
- Sketches
- Narrative

**Outputs:**
- Prototype designs
- Workflows
- Schemas

### 2. Story Architect Agent
**Purpose:** Extract founder stories, craft narratives

**Inputs:**
- Interviews
- Prompts

**Outputs:**
- Story frameworks
- Messaging
- Metaphors

### 3. Human Insight Agent
**Purpose:** Conduct emotional clarity sessions, analyze motivations

**Inputs:**
- Personal reflections
- Archetypes

**Outputs:**
- Insight summaries
- Guidance reports

### 4. Creative Mentor Agent
**Purpose:** Coach young creatives, generate exercises

**Inputs:**
- Skill level
- Goals

**Outputs:**
- Lessons
- Challenges
- Rituals

### 5. Acceleration Consultant Agent
**Purpose:** Identify bottlenecks in teams

**Inputs:**
- Team data
- Workflow info

**Outputs:**
- Audit reports
- Recommendations

## Expanded Agent Roles

### 6. Symbol Weaver Agent
**Purpose:** Creates symbolic visuals and meaning frameworks

**Capabilities:**
- Emotion to symbol conversion
- Visual metaphor creation
- Archetypal design elements

### 7. Ethical Clarity Agent
**Purpose:** Ensures emotional safety, transparency, and meaning alignment

**Responsibilities:**
- Monitor psychological safety
- Ensure trauma-informed practices
- Validate ethical alignment

### 8. Prototype Critic Agent
**Purpose:** Provides harsh but constructive critique to improve prototypes

**Focus:**
- Technical assessment
- User experience evaluation
- Narrative alignment check

### 9. Student Growth Agent
**Purpose:** Monitors progress in MVP School and gives personalized assignments

**Functions:**
- Track student progress
- Adapt learning paths
- Provide encouragement and challenges

### 10. Expression Agent
**Purpose:** Assists in converting emotions → symbols → art → tattoos → stories

**Outputs:**
- Symbolic designs
- Tattoo concepts
- Visual narratives
- Aesthetic direction
